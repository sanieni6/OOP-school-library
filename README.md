# 📖 OPP School Library with Ruby <a name="about-project"></a>

This is an easy-to-use solution to keep track of your library's inventory, including which books are available and who has borrowed them. Our app has got you covered, with features that allow you to:

Add new students and teachers
Add new books and their details
Save records of who has borrowed each book and when
With this app, you'll be able to keep accurate records and improve the borrowing experience for everyone. Students and teachers can easily find the books they need, and you can manage the inventory more efficiently.

Don't let library management become a headache. Try our app today and discover how easy it can be to manage your library like a pro!

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

- You'll need any code editor 

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com:sanieni6/OOP-school-library.git
  cd OOP-school-library
```

To run the project, execute the following command:

```sh
  main.rb
```
